ติดตั้ง Apps Script

เพิ่ม Backend API ใน Google Sheets (ทำครั้งเดียว)

🔧 ขั้นตอน:

  1. ใน Google Sheets → เมนู Extensions → Apps Script
  2. ลบโค้ดเดิมทั้งหมดออก
  3. คัดลอกโค้ดด้านล่าง → วางแทน → กด Save (Ctrl+S)
  4. เลือกฟังก์ชัน initSheets → กด ▶ Run (สร้าง Sheet ทั้งหมดอัตโนมัติ)
  5. คลิก Deploy → New deployment → Web app
  6. ตั้งค่า: Execute as = Me | Who can access = Anyone
  7. คัดลอก Web App URL ที่ได้
